PiHelper : VPN




Ce tuto est pour transformer votre Raspberry en Serveur VPN



1. Ressources


Connectez-vous en SSH à votre Raspberry Pi



II. Après mettez à jour votre OS



    sudo apt-get update
    sudo apt-get upgrade

III. Télécharger le script OPENVPN



    wget https://raw.githubusercontent.com/angristan/openvpn-install/master/openvpn-install.sh -O openvpn-install.sh
    sudo bash openvpn-install.sh

La plupart du temps, vous pouvez garder les options par défaut. Contentez de vous de cliquer sur « Entrée » si vous ne savez pas trop quoi répondre.
L’installation d’OpenVPN démarre. Quelques secondes plus tard, l’assistant vous demandera de créer votre premier utilisateur. Donnez-lui un nom (votre prénom ou le nom de l’appareil qui utilisera ce compte par exemple), et définissez le mot de passe si besoin (cela n’est pas indispensable).



Le serveur VPN est prêt !


Maintenant il faut ouvrir ses ports de box 1194 (regarder sur internet pour plus d'info)
